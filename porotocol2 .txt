MOD_GRAPH protocol


state handshake (рукопожатие)

После a:2 от админа клиент отвечает 2:n:min_ver;max_ver:resol_w;resol_h:
	n - размер данных
	min_ver - минимальная версия протокола поддерживаемая клиентом
	max_ver - максимальная версия протокола поддерживаемая клиентом
	resol_w/resol_h - ширина/высота картинки.


Дальше клиент ожидает ответа админа вида 
2:n:version:
по этой версии протокола будет осуществлятся обмен данными.
Дальше общение переходит в режим разговора, клиент сразу начинает посылать картинки


Пример рукопожатия:
> - сообщение от админа
< - сообщения от клиента

> a:2
< 2:12:1;2:1280;720:
> 2:1:2:


state talk (разговор)

(картинка разбивается на chislo пакетов размера 32KB (кроме последнего))
< 2:32000:nomer;chislo:image_data:
	nomer - номер пакета 

> 2:n:key_code:x_coord;y_coord:
	позиция курсора и код нажатой клавиши
